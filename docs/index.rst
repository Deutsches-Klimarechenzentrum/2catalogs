Welcome to 2catalogs Documentation
===================================

Overview
--------

**2catalogs** is a tool for generating catalogs for Earth System Model output.
It supports both STAC (SpatioTemporal Asset Catalog) and Intake catalog generation.

Features
--------

* Generate STAC catalogs from netCDF/Zarr datasets
* Generate Intake catalogs for data discovery
* Issue-based catalog generation workflow (Forge)
* Automated CI/CD pipeline integration
* Community catalog management
