## ğŸš€ Deployment

When you push to the `main` branch, the GitLab CI pipeline:

1. Copies entire `docs/` directory to `public/`
2. Copies `catalog/` directory to `public/catalog/`
3. Generates `documentation.html` with sidebar navigation
4. Deploys everything to GitLab Pages

## ğŸŒ URLs

After deployment, content is available at:

- **Main page**: `https://your-namespace.gitlab-pages.dkrz.de/-/2catalogs/`
- **Documentation**: `https://your-namespace.gitlab-pages.dkrz.de/-/2catalogs/documentation.html`
- **Specific docs**: Use URL hash like `documentation.html#docs/FORGE.md`
